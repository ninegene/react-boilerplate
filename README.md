## Setup
```
$ npm install gulp -g
$ git clone https://github.com/ninegene/react-boilerplate.git myproject
$ cd myproject
$ npm install
```

## Developing
Run dev http server and live reload browser on src files changes
```bash
$ gulp devServer
```
